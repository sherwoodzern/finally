    - `execute_trade(conn, cache, ticker, side, quantity, user_id="default") -> TradeResponse`:
      1. Uppercase + strip ticker.
      2. SELECT 1 FROM watchlist WHERE user_id = ? AND ticker = ? — if None, raise `UnknownTicker`.
      3. `price = cache.get_price(ticker)` — if None, raise `PriceUnavailable`.
      4. SELECT cash_balance FROM users_profile WHERE id = ?; SELECT id, quantity, avg_cost FROM positions WHERE user_id=? AND ticker=?.
      5. For BUY: if `quantity * price > cash_balance` raise `InsufficientCash(f"Need ${gross:.2f}, have ${cash_balance:.2f}")`; else compute `new_qty = old_qty + quantity`, `new_avg = ((old_qty * old_avg) + (quantity * price)) / new_qty`, `new_cash = cash_balance - gross`.
      6. For SELL: if `quantity > old_qty + 1e-9` raise `InsufficientShares(f"Requested {quantity}, held {old_qty}")`; else `new_qty = old_qty - quantity`, `new_avg = old_avg`, `new_cash = cash_balance + gross`.
      7. UPDATE users_profile SET cash_balance = ?.
      8. If `abs(new_qty) < 1e-9` → `DELETE FROM positions WHERE user_id=? AND ticker=?` (D-15) and set `pos_qty_out, pos_avg_out = 0.0, 0.0`.
         Elif `pos_row is None` → INSERT new positions row with a fresh uuid.
         Else → UPDATE existing positions row.
      9. INSERT trade row into `trades` with fresh uuid.
      10. INSERT snapshot row into `portfolio_snapshots` with `total_value = _compute_total_value_with(conn, cache, new_cash, user_id)`.
      11. `conn.commit()` exactly once.
      12. Log `logger.info("Trade executed: %s %s x %.4f @ %.2f (cash=%.2f)", ticker, side, quantity, price, new_cash)`.
      13. Return a `TradeResponse(...)`.
    - On any raise, ZERO writes have occurred (validate-then-write per D-12).

           conn: sqlite3.Connection,
           cache: PriceCache,
           user_id: str = DEFAULT_USER_ID,
       ) -> PortfolioResponse:
           """Return cash, total_value, and positions with live prices falling back to avg_cost."""
           cash_row = conn.execute(
               "SELECT cash_balance FROM users_profile WHERE id = ?", (user_id,)
           ).fetchone()
           cash = float(cash_row["cash_balance"]) if cash_row else 0.0

           rows = conn.execute(
               "SELECT ticker, quantity, avg_cost FROM positions WHERE user_id = ? "
               "ORDER BY ticker ASC",
               (user_id,),
           ).fetchall()

           positions: list[PositionOut] = []

               qty = float(row["quantity"])
               avg = float(row["avg_cost"])
               cached = cache.get_price(ticker)
               current = cached if cached is not None else avg

               pct = ((current - avg) / avg * 100.0) if avg != 0.0 else 0.0
               total += qty * current
               positions.append(PositionOut(
                   ticker=ticker, quantity=qty, avg_cost=avg,
                   current_price=current, unrealized_pnl=round(pnl, 2),
                   change_percent=round(pct, 2),
               ))

           return PortfolioResponse(
               cash_balance=cash,
               total_value=round(total, 2),
               positions=positions,
           )

           """Zero-arg tick observer that writes a snapshot every 60s (D-05, D-06, D-07).

             - state.db:                sqlite3.Connection
             - state.price_cache:       PriceCache
             - state.last_snapshot_at:  float (monotonic clock)
           Uses time.monotonic() for the 60s threshold; datetime.now(UTC).isoformat() for
           the recorded_at column value (Pitfall 6).

           def observer() -> None:
               now = time.monotonic()

                   return

               state.db.execute(
                   "INSERT INTO portfolio_snapshots (id, user_id, total_value, recorded_at) "
                   "VALUES (?, ?, ?, ?)",
                   (str(uuid.uuid4()), DEFAULT_USER_ID, total_value, datetime.now(UTC).isoformat()),
               )
               state.db.commit()
               state.last_snapshot_at = now
