## Critical Anti-Patterns (do NOT repeat these)

- **Worktree base mismatch (advisory):** When a `gsd-executor` worktree agent reports its base is wrong and the sandbox blocks `git reset --hard`, do not loop on retries. Fall through to inline execution.
- **Sequential subagent Write denial (advisory):** If a non-worktree `gsd-executor` reports Write/mkdir permission denials, fall through to inline execution from the orchestrator. The orchestrator's own Write/Edit/Bash tools work fine.
- **Lightweight Charts v4 API drift:** `chart.addLineSeries(...)` is removed in v5. Use `chart.addSeries(LineSeries, {...})` with `LineSeries` imported as a value token.
- **`detail.code` typo:** The backend error body keys on `detail.error`, not `detail.code`. CONTEXT.md is informal wording; backend code is authoritative.
