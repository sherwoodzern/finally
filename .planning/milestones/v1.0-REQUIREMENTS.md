## Out of Scope (v1.0)

| Feature | Reason |
|---------|--------|
| Limit orders, stop orders, partial fills, order book | Massive complexity with no demo value; market orders only. |
| Fees, commissions, slippage, borrow costs | Simulated portfolio; instant fill at cached mid price. |
| Real-money brokerage integration | Fake money by design — safe playground. |
| WebSockets for price streaming | SSE one-way push is all the price stream needs. |
| Postgres or external DB server | Single-file SQLite + Docker volume is enough for single-user localhost. |
| Trade confirmation dialogs | Deliberately omitted — AI-driven trade execution must feel immediate and agentic. |
